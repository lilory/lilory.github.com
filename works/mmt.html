마이뮤직테이스트

팬 중심 크라우드 콘서트 메이킹 플랫폼을 만드는 스타트업 마이뮤직테이스트 http://mymusictaste.com/에서 서비스 관리를 위한 데이터 관리 페이지의 UX설계부터 프론트엔드 개발까지 1인 프로젝트로 진행했습니다.

Role

프론트엔드 개발, UX

2013-2014

배경

서비스 런칭 후 사용자가 유입되면서 사내에서 사용자 파악 및 콘서트 요청 데이터 분석에 대한 니즈가 발생하여 필요한 데이터를 간단하게 조회할 수 있는 데이터 분석 페이지 제작을 고민하게 되었습니다.

과정

·사내 데이터 분석 니즈 조사
·필요한 기능 정의
·페이지 UX설계 및 프론트엔드 개발
·지속적인 개선을 진행

결과

·가입 및 기능별 사용 기록 등의 User behavior 조회 기능
·사용자의 Demographics, Geolocation 등 조회 기능
·사용자 유입 경로 등의 User acquisition 조회 기능
·아티스트별, 도시별 등 콘서트 요청 데이터 분석 기능
